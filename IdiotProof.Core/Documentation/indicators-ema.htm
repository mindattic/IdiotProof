EMA Indicators

Exponential Moving Averages (EMAs) give more weight to recent price data, making them more responsive to new information than simple moving averages.

What is EMA?

The Exponential Moving Average is a type of moving average that places greater weight on the most recent data points. Unlike a Simple Moving Average (SMA) which weights all values equally, EMA reacts faster to price changes.

EMA CALCULATION ═══════════════════════════════════════════════════════════════════════════════ EMA = (Price × Multiplier) + (Previous EMA × (1 - Multiplier)) Where: Multiplier = 2 / (Period + 1) Example for EMA(9): Multiplier = 2 / (9 + 1) = 0.2 If Price = $50.00, Previous EMA = $49.50: EMA = ($50.00 × 0.2) + ($49.50 × 0.8) = $10.00 + $39.60 = $49.60 COMMON EMA PERIODS ═══════════════════════════════════════════════════════════════════════════════ ┌──────────┬─────────────────────┬─────────────────────────────────────┐ │ Period │ Trading Style │ Purpose │ ├──────────┼─────────────────────┼─────────────────────────────────────┤ │ EMA(9) │ Scalping/Day │ Very short-term trend │ │ EMA(21) │ Day/Swing │ Short-term trend │ │ EMA(50) │ Swing │ Medium-term trend │ │ EMA(200)│ Position │ Long-term trend (major support) │ └──────────┴─────────────────────┴─────────────────────────────────────┘

The EMA Stack

IdiotProof uses an "EMA Stack" concept where the relationship between multiple EMAs indicates trend strength and direction.

EMA STACK ALIGNMENT ═══════════════════════════════════════════════════════════════════════════════ BULLISH STACK: Price > EMA(9) > EMA(21) > EMA(50) Price ───────────────────────────────────────────────────────── ↗ EMA(9) ───────────────────────────────────────────────── ↗ EMA(21) ───────────────────────────────────────────── ↗ EMA(50) ───────────────────────────────────────── All EMAs stacked and rising = Strong uptrend BEARISH STACK: Price < EMA(9) < EMA(21) < EMA(50) EMA(50) ───────────────────────────────────────── ↘ EMA(21) ───────────────────────────────────────────── ↘ EMA(9) ───────────────────────────────────────────────── ↘ Price ───────────────────────────────────────────────────────── All EMAs stacked downward = Strong downtrend CHOPPY/TRANSITIONAL: EMAs Converging ────────────────────────╲ ────────────────────────────╲ ← EMAs converging ────────────────────────────────╲ ▲ │ No clear trend │ Wait for separation

EMA IdiotScript Commands

IsEmaAbove(period)

Returns true when current price is above the specified EMA.

Ticker(AAPL)
    .IsEmaAbove(9)   // Price above EMA(9)
    .IsEmaAbove(21)  // AND price above EMA(21)
    .Entry(150)
    .TakeProfit(155)

IsEmaBelow(period)

Returns true when current price is below the specified EMA.

Ticker(SPY)
    .IsEmaBelow(21)  // Price below EMA(21)
    .Order(IS.SHORT)
    .TakeProfit(495)
    .StopLoss(508)

IsEmaBetween(shortPeriod, longPeriod)

Returns true when price is between two EMAs. This is the pullback zone - useful for finding entries in trends. 

Ticker(NVDA)
    .IsEmaBetween(9, 21)  // Price between EMA(9) and EMA(21)
    .IsMomentumAbove(0)     // Positive momentum
    .Entry(900)
    .TakeProfit(920)

IsEmaTurningUp(period)

Returns true when the EMA slope is turning positive or flat after a decline. This catches early trend reversals. 

Ticker(TSLA)
    .IsEmaTurningUp(9)   // EMA(9) slope turning up
    .IsAboveVwap()          // Confirm with VWAP
    .Entry(250)
    .TakeProfit(265)

EMA in Market Score

EMA contributes 15% to the total market score. The score evaluates the EMA stack alignment:

EMA SCORE CONTRIBUTION ═══════════════════════════════════════════════════════════════════════════════ EMA Stack Condition Score Contribution ───────────────────────────────────────────────────────── Price > EMA(9) > EMA(21) > EMA(50) → +100 (max bullish) Price > EMA(9) > EMA(21) → +67 Price > EMA(9) → +33 Price ≈ EMA(9) → 0 (neutral) Price < EMA(9) → -33 Price < EMA(9) < EMA(21) → -67 Price < EMA(9) < EMA(21) < EMA(50) → -100 (max bearish) Additional Factors: ───────────────────────────────────────────────────────── EMA slopes all rising → +20 bonus EMA slopes all falling → -20 penalty EMAs converging tightly → Score dampened (uncertainty)
Condition Score Interpretation
Full bullish stack + rising slopes +100 Strong uptrend, highest conviction
Price above 9 and 21 EMA +67 Healthy uptrend
Price above 9 EMA only +33 Short-term bullish, watch for continuation
EMAs tangled, no clear order -20 to +20 Choppy, avoid or reduce size
Full bearish stack + falling slopes -100 Strong downtrend, short or avoid

EMA Warm-Up Requirements

⚠️ Important: EMA Warm-Up

EMAs require historical bars equal to their period before they become accurate. Start the backend early to collect enough bars.

EMA Period Bars Needed Start Early By
EMA(9) 9 bars 10 minutes
EMA(21) 21 bars 25 minutes
EMA(50) 50 bars 1 hour
EMA(200) 200 bars 3+ hours

Recommendation: For most day trading strategies using EMA(9), EMA(21), and EMA(50), start the backend 30 minutes before you plan to trade.

EMA Trading Strategies

1. Bullish EMA Stack

Enter when all EMAs are properly stacked bullish.

Ticker(AAPL)
    .Name("EMA Stack Long")
    .Session(IS.RTH)
    .Quantity(50)
    .IsEmaAbove(9)        // Price > EMA(9)
    .IsEmaAbove(21)       // Price > EMA(21)
    .IsEmaAbove(50)       // Price > EMA(50)
    .IsAboveVwap()         // Confirm with VWAP
    .Entry(150)
    .TakeProfit(158)
    .StopLoss(147)
    .AdaptiveOrder(IS.AGGRESSIVE)

2. EMA Pullback Entry

Enter on pullback to the 9/21 EMA zone in an uptrend.

Ticker(NVDA)
    .Name("EMA Pullback")
    .Session(IS.RTH)
    .Quantity(20)
    .IsEmaBetween(9, 21)   // In the pullback zone
    .IsEmaAbove(50)         // Above major support
    .IsEmaTurningUp(9)      // Short EMA turning up
    .IsMomentumAbove(0)      // Positive momentum
    .Entry(900)
    .TakeProfit(930)
    .StopLoss(885)

3. EMA Trend Reversal

Catch early trend changes with EMA slope detection.

Ticker(TSLA)
    .Name("EMA Reversal")
    .Session(IS.RTH)
    .Quantity(15)
    .IsEmaTurningUp(9)      // EMA(9) slope turning positive
    .IsEmaTurningUp(21)     // EMA(21) also turning
    .IsRsiOversold(35)      // Coming out of oversold
    .Entry(250)
    .TakeProfit(270)
    .StopLoss(240)

4. Bearish EMA Short

Short when EMAs are stacked bearish.

Ticker(SPY)
    .Name("EMA Stack Short")
    .Session(IS.RTH)
    .Quantity(100)
    .IsEmaBelow(9)         // Price < EMA(9)
    .IsEmaBelow(21)        // Price < EMA(21)
    .IsBelowVwap()          // Below VWAP
    .IsDiNegative()         // -DI > +DI
    .Order(IS.SHORT)
    .TakeProfit(490)
    .StopLoss(508)

EMA Trading Tips

💡 Pro Tips
  • Use multiple EMAs together - Stack alignment is more reliable than single EMA
  • EMA(9) is fast, EMA(21) is the trend filter - Use 9 for entry timing, 21 for trend direction
  • Pullbacks to EMA(21) are high-probability entries in strong trends
  • EMA(50) acts as major support/resistance - Expect bounces or breaks
  • Combine with momentum indicators - EMA + MACD or RSI for confirmation
⚠️ Common Mistakes
  • Trading EMA crossovers in choppy markets - wait for confirmation
  • Using EMA without allowing warm-up time
  • Ignoring EMA slope - flat EMAs mean indecision, not trend
  • Fighting the EMA stack - don't long when all EMAs bearish
