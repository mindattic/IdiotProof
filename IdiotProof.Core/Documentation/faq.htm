Frequently Asked Questions

Answers to common questions about IdiotProof trading system.

General Questions

What is IdiotProof?

IdiotProof is an automated trading system that connects to Interactive Brokers TWS to execute stock trades based on strategies defined in IdiotScript, a domain-specific language. It uses AI/ML components including LSTM neural networks, pattern matching, and ChatGPT integration for enhanced decision-making.

What broker does IdiotProof support?

Currently, IdiotProof only supports Interactive Brokers via the TWS (Trader Workstation) API. TWS must be running and configured to accept API connections on port 7497 (paper trading) or 7496 (live).

Can I use IdiotProof for paper trading?

Yes! This is highly recommended before live trading. Connect to TWS paper trading account (port 7497) to test strategies without risk.

What markets can I trade?

IdiotProof is designed for US equities. It supports:

  • NYSE and NASDAQ stocks
  • Premarket (4:00 AM - 9:30 AM ET)
  • Regular Trading Hours (9:30 AM - 4:00 PM ET)
  • After Hours (4:00 PM - 8:00 PM ET)

IdiotScript Questions

What's the difference between Entry() and Order()?

Entry(price) is a CONDITION that triggers when price reaches a level. Order(IS.LONG) is an ACTION that executes when all conditions are met.

// Entry = price trigger condition Entry(150) // Wait for price >= 150 // Order = what to do when conditions met Order(IS.LONG) // Execute a buy order

Do I need parentheses on commands like IsAboveVwap?

Yes. While the parser accepts IsAboveVwap for backwards compatibility, the canonical form is IsAboveVwap(). Always use parentheses.

What happens if conditions are never met?

The strategy waits. No order is placed until ALL entry conditions are satisfied. The strategy can wait indefinitely until conditions are met or the session ends.

How do I exit at the end of the day?

Use ExitStrategy(IS.BELL) to exit 1 minute before session close. Add .IsProfitable() to only exit if the position is profitable.

Trading Mode Questions

What's the difference between normal strategies and AutonomousTrading?

Aspect Normal Strategy AutonomousTrading
Entry Conditions You define using IdiotScript AI decides based on Market Score
Exit Conditions TakeProfit/StopLoss you set ATR-based, AI adjusts
Direction Fixed per strategy Can flip long↔short
Control Full control Hands-off

Should I use AdaptiveOrder or AutonomousTrading?

AdaptiveOrder: Use when you have specific entry conditions but want smart exit management. You control when to enter, system adjusts TP/SL dynamically.

AutonomousTrading: Use when you want the AI to handle everything. Just specify the ticker and let it trade.

Indicator Questions

Why aren't my indicator conditions triggering?

Most likely: indicators need warm-up time. Indicators require historical data (candles) before they can calculate. Start the backend early:

  • EMA(9): Start 10 minutes before trading
  • EMA(21): Start 25 minutes before
  • EMA(200): Start 3+ hours before
  • ADX(14): Start 30 minutes before

What is Market Score?

Market Score is a composite value from -100 (extremely bearish) to +100 (extremely bullish). It combines 11 indicators with weighted contributions. See Market Score for details.

How does LSTM affect the Market Score?

The LSTM predictor can adjust the Market Score by up to ±25 points based on its direction prediction and confidence level. This helps the score account for patterns that simple indicators might miss.

AI & Machine Learning Questions

Does IdiotProof require an internet connection?

For basic trading: No (after connecting to TWS locally).
For AI Advisor (ChatGPT): Yes, requires internet and valid OpenAI API key.

How much does the AI Advisor cost?

The AI Advisor uses OpenAI's API. Costs depend on usage. Each analysis call uses approximately 500-1000 tokens (~$0.01-0.03 per call with GPT-4).

Can the LSTM predict the future?

No. The LSTM provides probabilistic directional guidance based on learned patterns. It's one input among many - it can be wrong. Never rely solely on LSTM predictions.

How does the system learn over time?

Three learning mechanisms:

  1. Ticker Profiles: Win rates and optimal thresholds per symbol
  2. Indicator Weights: Which indicators work best per symbol
  3. Pattern History: Stored patterns improve future matching

Order Execution Questions

Are orders placed as market or limit?

By default, orders are market orders for immediate execution. You can specify limit orders using .OrderType(OrderType.Limit).

Can I trade outside regular hours?

Yes. Use .Session(IS.PREMARKET) or .Session(IS.AFTERHOURS). Also ensure .OutsideRTH() is set for the order to execute.

What happens if TWS disconnects?

Open orders remain on IBKR servers. IdiotProof attempts reconnection. Upon reconnect, it syncs position state. Stop losses placed at IBKR continue to protect your position.

Risk Management Questions

Will IdiotProof prevent losses?

No. Trading involves risk. IdiotProof provides tools like stop losses and adaptive orders, but losses are still possible and expected. Never trade more than you can afford to lose.

What's the emergency exit?

AdaptiveOrder has an emergency exit when Market Score drops below threshold (default -70). This closes the position immediately regardless of profit/loss.

Can I set a maximum loss per day?

This feature is planned but not yet implemented. Currently, set stop losses on each trade individually. Consider using IBKR's built-in daily loss limits.
