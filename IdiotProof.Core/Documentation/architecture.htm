Strategy State Machine

STRATEGY EXECUTION STATE MACHINE ═══════════════════════════════════════════════════════════════════════════════ ┌─────────────────────────────────────────────────────────────────────────┐ │ │ │ ┌──────────┐ Conditions ┌──────────┐ Order ┌──────────┐│ │ │ │ Met │ │ Filled │ ││ │ │ IDLE ├─────────────────►│ ENTRY ├─────────────►│ POSITION ││ │ │ │ │ PENDING │ │ OPEN ││ │ └──────────┘ └──────────┘ └────┬─────┘│ │ ▲ │ │ │ │ │ │ │ │ Reset ┌─────────────────────────────────────────┘ │ │ │ (Repeat) │ │ │ │ ▼ │ │ ┌────┴─────┐ ┌──────────┐ │ │ │ │ │ │ │ │ │COMPLETED │◄─┤ EXIT │◄─── TP/SL/TSL/ExitStrategy triggered │ │ │ │ │ │ │ │ └──────────┘ └──────────┘ │ │ │ └─────────────────────────────────────────────────────────────────────────┘ CONDITION EVALUATION (Sequential State Machine) ═══════════════════════════════════════════════════════════════════════════════ Example: IsGapUp(5).IsAboveVwap().IsEmaAbove(9).Order() State 0: Wait for gap up 5%+ └─► Gap detected? → Move to State 1 State 1: Wait for price above VWAP └─► Above VWAP? → Move to State 2 State 2: Wait for price above EMA(9) └─► Above EMA? → Move to State 3 State 3: ALL CONDITIONS MET └─► Execute Order() Key: Each condition must be met IN SEQUENCE. Conditions are evaluated as a state machine over time, NOT as a simultaneous AND condition.
