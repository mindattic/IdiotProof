Ticker Learning System

The system learns what works for each specific stock. Every trade is recorded, analyzed, and used to improve future entries, exits, and timing. Over time, the system becomes specialized for each ticker.

How It Works

What Gets Stored

Each ticker has a profile stored in Profiles/{SYMBOL}.json. The profile contains:

TradeRecord - Per-Trade Data

Field Description
EntryTime When the trade was entered
ExitTime When the trade was exited
EntryPrice Price at entry
ExitPrice Price at exit
IsLong True for long, false for short
Quantity Number of shares
EntryScore Market score at entry
Entry*Score Individual indicator scores (VWAP, EMA, RSI, MACD, ADX, Volume, Bollinger, Stochastic, OBV, CCI, WilliamsR)
RsiAtEntry Raw RSI value at entry
AdxAtEntry Raw ADX value at entry
ExitReason "TP", "SL", "score", or "END"
TimeBucket 15-min bucket like "10:15"

TickerProfile - Aggregate Statistics

Category Fields
Overall Stats TotalTrades, TotalWins, TotalLosses, WinRate, ProfitFactor, TotalPnL
Long Stats LongTrades, LongWins, LongWinRate
Short Stats ShortTrades, ShortWins, ShortWinRate
Streak Tracking CurrentStreak, LongestWinStreak, LongestLossStreak
Threshold Stats Win rate & expectancy at each score bucket (5-point intervals)
Time Windows Win rate & average profit for each 15-min time bucket
Indicator Correlations Which indicator conditions lead to wins
Optimal Thresholds Learned best entry/exit score levels

Threshold Learning

The system learns what entry score works best for each ticker by tracking win rates at different threshold levels.

THRESHOLD BUCKET ANALYSIS ═══════════════════════════════════════════════════════════════════════════════ Example: NVDA Long Entry Threshold Stats Score Bucket │ Trades │ Wins │ Win Rate │ Expectancy │ Profit Factor ─────────────┼────────┼──────┼──────────┼────────────┼─────────────── 60 │ 12 │ 5 │ 42% │ -$15 │ 0.8 65 │ 18 │ 10 │ 56% │ +$28 │ 1.4 70 │ 15 │ 9 │ 60% │ +$45 │ 1.6 75 │ 11 │ 8 │ 73% │ +$82 │ 2.1 ◄── Best! 80 │ 6 │ 5 │ 83% │ +$65 │ 2.5 85 │ 2 │ 2 │ 100% │ +$40 │ ∞ └─ Too few trades to trust RESULT: OptimalLongEntryThreshold = 75 (Highest expectancy with sufficient sample size) CONFIDENCE BLENDING ═══════════════════════════════════════════════════════════════════════════════ Confidence = min(100, TotalTrades × 2) Blended Threshold = Default × (1 - Confidence%) + Learned × Confidence% Example with 26 trades (52% confidence): ───────────────────────────────────────────────────────────────────────── Default threshold: 70 Learned optimal: 75 Confidence: 52% Blended = 70 × 0.48 + 75 × 0.52 = 33.6 + 39.0 = 72.6 (rounded to 73) The system enters at 73 instead of 70, using the learned insight that this ticker performs better with a slightly higher threshold.

Time Window Analysis

The system learns which times of day work best for each ticker. Time is bucketed into 15-minute windows.

Indicator Correlation Learning

The system tracks which indicator conditions correlate with winning trades for each ticker. This reveals what signals work best for specific stocks.

Streak Awareness

The system adjusts behavior based on recent win/loss streaks:

Streak Risk Multiplier Effect
3+ losses in a row 0.5 Halve position size
2 losses in a row 0.75 Reduce position size 25%
Normal 1.0 Standard position size
5+ wins in a row 0.9 Slight reduction to protect gains
// In trading logic double riskMultiplier = profile.GetStreakRiskMultiplier(); int adjustedQuantity = (int)(baseQuantity * riskMultiplier);

Using Learned Data

Get Adjusted Thresholds

// Load profile var profile = TickerProfileManager.Load("NVDA"); // Get threshold blended with learned optimal int adjustedLongEntry = profile.GetAdjustedLongEntryThreshold(DefaultThreshold); // Check if current time is good for trading bool goodTime = profile.IsGoodTimeWindow(DateTime.Now); // Get position size adjustment based on streak double riskMult = profile.GetStreakRiskMultiplier();

Record a New Trade

var trade = new TradeRecord { EntryTime = entryTime, ExitTime = exitTime, EntryPrice = 150.25, ExitPrice = 152.50, IsLong = true, Quantity = 100, EntryScore = 72, RsiAtEntry = 42.5, AdxAtEntry = 38.2, ExitReason = "TP" // ... all other indicator scores }; profile.RecordTrade(trade); TickerProfileManager.Save(profile);

Get Best Indicator Conditions

// Find what works for this ticker var bestConditions = profile.GetBestIndicatorConditions(minOccurrences: 5, count: 10); foreach (var cond in bestConditions) { Console.WriteLine($"{cond.IndicatorName} {cond.Condition}: {cond.WinRate:F1}% win rate"); }
