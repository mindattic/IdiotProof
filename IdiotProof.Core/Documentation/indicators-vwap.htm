VWAP Indicators

Volume Weighted Average Price (VWAP) is a trading benchmark that gives the average price a security has traded at throughout the day, weighted by volume.

What is VWAP?

VWAP represents the fair value of a stock for the trading day. Institutional traders use VWAP to gauge whether they received a good fill price. Price above VWAP suggests bullish sentiment; below VWAP suggests bearish.

VWAP CALCULATION ═══════════════════════════════════════════════════════════════════════════════ VWAP = Cumulative(Price × Volume) / Cumulative(Volume) Where Price = (High + Low + Close) / 3 (Typical Price) Example: ┌─────────┬────────┬────────┬────────┬─────────┬──────────────┐ │ Time │ High │ Low │ Close │ Volume │ Typical × Vol│ ├─────────┼────────┼────────┼────────┼─────────┼──────────────┤ │ 9:30 │ $50.50 │ $50.00 │ $50.25 │ 10,000 │ $502,500 │ │ 9:31 │ $50.60 │ $50.20 │ $50.40 │ 15,000 │ $756,000 │ │ 9:32 │ $50.80 │ $50.30 │ $50.55 │ 12,000 │ $606,600 │ └─────────┴────────┴────────┴────────┴─────────┴──────────────┘ Cumulative P×V = $502,500 + $756,000 + $606,600 = $1,865,100 Cumulative Vol = 10,000 + 15,000 + 12,000 = 37,000 VWAP = $1,865,100 / 37,000 = $50.41

VWAP IdiotScript Commands

IsAboveVwap()

Returns true when current price is above VWAP. Basic bullish filter.

Ticker(AAPL)
    .IsAboveVwap()  // Only enter when price > VWAP
    .Entry(150)
    .TakeProfit(155)

IsBelowVwap()

Returns true when current price is below VWAP. Basic bearish filter.

Ticker(AAPL)
    .IsBelowVwap()  // Only short when price < VWAP
    .Entry(145)
    .Order(IS.SHORT)
    .TakeProfit(140)

IsCloseAboveVwap()

Returns true when the candle closes above VWAP. This is a stronger signal than IsAboveVwap because it confirms the price held above VWAP by the end of the candle. 

Ticker(NVDA)
    .IsCloseAboveVwap()  // Stronger confirmation
    .IsEmaAbove(9)
    .Entry(900)
    .TakeProfit(920)

IsVwapRejection()

Returns true when price wicks above VWAP but closes below. This is a bearish rejection signal indicating sellers defended VWAP. 

Ticker(SPY)
    .IsVwapRejection()  // Bearish rejection pattern
    .Order(IS.SHORT)
    .TakeProfit(495)
    .StopLoss(510)

VWAP in Market Score

VWAP contributes 10% to the total market score. The score is calculated based on how far price is from VWAP:

VWAP SCORE CONTRIBUTION ═══════════════════════════════════════════════════════════════════════════════ Distance from VWAP Score Contribution ───────────────────────────────────────────── 5%+ above VWAP → +100 (max bullish) 2.5% above VWAP → +50 At VWAP (±0.5%) → 0 (neutral) 2.5% below VWAP → -50 5%+ below VWAP → -100 (max bearish) Formula: ──────────────────────────────────────────────────────────── percentFromVwap = ((price - vwap) / vwap) × 100 score = Clamp(percentFromVwap × 20, -100, +100) Example: Price $52, VWAP $50 percentFromVwap = ((52 - 50) / 50) × 100 = 4% score = 4 × 20 = +80
Condition Score Range Interpretation
Price 3%+ above VWAP +60 to +100 Strong bullish - extended from fair value
Price 1-3% above VWAP +20 to +60 Moderately bullish
Price near VWAP (±1%) -20 to +20 Neutral - at fair value
Price 1-3% below VWAP -60 to -20 Moderately bearish
Price 3%+ below VWAP -100 to -60 Strong bearish - discounted from fair value

VWAP Trading Strategies

1. VWAP Bounce (Long)

Enter long when price pulls back to VWAP and bounces.

Ticker(AAPL)
    .Name("VWAP Bounce Long")
    .Session(IS.RTH)
    .Quantity(50)
    .IsCloseAboveVwap()     // Confirmed above VWAP
    .IsEmaAbove(9)          // Short-term trend up
    .IsHigherLows()         // Building support
    .Entry(150)
    .TakeProfit(155)
    .StopLoss(148)
    .AdaptiveOrder(IS.BALANCED)

2. VWAP Rejection (Short)

Short when price fails to hold above VWAP.

Ticker(SPY)
    .Name("VWAP Rejection Short")
    .Session(IS.RTH)
    .Quantity(100)
    .IsVwapRejection()     // Bearish rejection
    .IsEmaBelow(21)         // Medium-term trend down
    .Order(IS.SHORT)
    .TakeProfit(495)
    .StopLoss(508)
    .AdaptiveOrder(IS.CONSERVATIVE)

3. VWAP Reclaim

Enter when price reclaims VWAP after being below.

Ticker(TSLA)
    .Name("VWAP Reclaim")
    .Session(IS.RTH)
    .Quantity(20)
    .IsCloseAboveVwap()     // Just reclaimed VWAP
    .IsMomentumAbove(0)     // Positive momentum
    .IsVolumeAbove(1.2)     // Volume confirmation
    .Entry(250)
    .TakeProfit(265)
    .StopLoss(245)

Warm-Up Requirements

ℹ️ VWAP Warm-Up

VWAP requires minimal warm-up because it calculates from the first bar of the session. However, VWAP is most reliable after 30+ minutes of trading when enough volume has accumulated.

Time After Open VWAP Reliability Recommendation
0-15 minutes Low Avoid VWAP signals - too volatile
15-30 minutes Moderate Use with caution, confirm with other indicators
30+ minutes High VWAP is reliable benchmark

VWAP Trading Tips

💡 Pro Tips
  • VWAP resets daily - It only applies to the current session
  • Combine with volume - VWAP moves are more significant on high volume
  • Watch for VWAP magnets - Price often returns to VWAP
  • Use IsCloseAboveVwap over IsAboveVwap for stronger signals
  • VWAP works best in RTH - Extended hours have less volume
⚠️ Common Mistakes
  • Trading VWAP signals in the first 15 minutes of market open
  • Ignoring volume confirmation on VWAP breaks
  • Using VWAP in pre-market or after-hours (low reliability)
  • Not combining VWAP with trend indicators (EMA, ADX)
