Quick Start Guide

Get up and running with IdiotProof in 10 minutes. This guide covers prerequisites, configuration, and your first trade.

Prerequisites

ℹ️ What You Need
  • Interactive Brokers Account - Live or Paper Trading
  • TWS or IB Gateway - Running with API enabled
  • .NET 10 Runtime - For the backend application
  • OpenAI API Key - Optional, for AI Advisor feature

Configure TWS API Settings

TWS CONFIGURATION CHECKLIST ═══════════════════════════════════════════════════════════════════════════════ In TWS: File → Global Configuration → API → Settings [*] Enable ActiveX and Socket Clients [*] Read-Only API (uncheck for live trading) [ ] Download open orders on connection Socket port: 7496 (live) or 7497 (paper) Trusted IP Addresses: ┌─────────────────────────┐ │ 127.0.0.1 │ ← Required for local connection └─────────────────────────┘ In TWS: File → Global Configuration → API → Precautions [ ] Bypass Order Precautions for API Orders ↑ Check this for automated trading (understand the risks!)

Step 1: Configure Settings

Create or edit appsettings.json in the application folder:

{
  "TwsHost": "127.0.0.1",
  "TwsPort": 7497,           // 7497=paper, 7496=live
  "ClientId": 1,
  "OpenAIApiKey": "sk-...",   // Optional
  "MaxCandlesticks": 500,
  "DataFolder": "Data"
}

Step 2: Choose Your Trading Mode

IdiotProof supports two main trading modes:

📜
Manual Strategy Mode
Define specific conditions, entry prices, and exit targets. You control exactly when and how trades execute. Use Entry(), IsAboveVwap(), etc.
🤖
Autonomous Mode
Let the AI handle everything. Just specify ticker and quantity. System calculates entries, exits, and adjusts dynamically. Use AutonomousTrading().

Step 3: Create Your First Strategy

Option A: Manual Strategy

// A simple breakout strategy for NVDA
Ticker(NVDA)
    .Name("NVDA Breakout")
    .Session(IS.RTH)         // Regular Trading Hours only
    .Quantity(5)            // Trade 5 shares
    .Entry(850)             // Trigger when price hits $850
    .IsAboveVwap()          // Price must be above VWAP
    .IsEmaAbove(9)          // Price above 9-period EMA
    .Order(IS.LONG)         // Buy order
    .TakeProfit(870)        // Exit at $870
    .StopLoss(840)          // Stop loss at $840

Option B: Autonomous Strategy

// Let AI handle everything
Ticker(AAPL)
    .Name("AAPL Autonomous")
    .Session(IS.RTH)
    .Quantity(10)
    .AutonomousTrading()    // AI manages everything

Option C: Use the Watchlist

For multiple tickers, edit Data/watchlist.json:

{
  "description": "Morning Trading Watchlist",
  "enabled": true,
  "session": "RTH",
  "tickers": [
    { "symbol": "NVDA", "quantity": 5, "enabled": true },
    { "symbol": "AAPL", "quantity": 10, "enabled": true },
    { "symbol": "TSLA", "quantity": 3, "enabled": false }
  ]
}

Step 4: Understand Indicator Warm-Up

🚨 Critical: Start Early!

Technical indicators need historical data to calculate. If you start the backend right at market open, indicators won't be ready and the system will miss early opportunities or make poor decisions.

Indicator Bars Needed Start Early By
EMA(9) 9 bars 10 minutes
EMA(21) 21 bars 25 minutes
ADX(14) / DI 28 bars 30 minutes
RSI(14) 15 bars 20 minutes
MACD(12,26,9) 35 bars 40 minutes
EMA(200) 200 bars 3+ hours

Recommended Start Times

Step 5: Start the Backend

# Navigate to the solution folder
cd IdiotProof

# Build the project
dotnet build IdiotProof.Core/IdiotProof.Core.csproj

# Run the backend
dotnet run --project IdiotProof.Core/IdiotProof.Core.csproj
