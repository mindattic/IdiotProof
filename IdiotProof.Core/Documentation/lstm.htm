LSTM Neural Network

Long Short-Term Memory networks for price direction prediction, capturing temporal dependencies in sequential market data.

What is LSTM?

LSTM (Long Short-Term Memory) is a type of recurrent neural network designed to learn long-term dependencies in sequential data. Unlike traditional neural networks, LSTM can "remember" patterns over many time steps - perfect for analyzing price action that depends on historical context.

LSTM vs TRADITIONAL NEURAL NETWORK ═══════════════════════════════════════════════════════════════════════════════ TRADITIONAL NN (Memoryless) LSTM (Has Memory) Input ──► [Hidden] ──► Output Input ──► [Cell State + Hidden] ──► Output ↑ │ └─────────┘ (Memory loop) Problem: Each prediction is Solution: Each prediction uses independent. FORGET GATE: f(t) = σ(Wf · [h(t-1), x(t)] + bf) Decides what info to throw away from cell state Value 0 = forget completely, 1 = keep completely 2. INPUT GATE: i(t) = σ(Wi · [h(t-1), x(t)] + bi) Decides which values to update 3. CELL UPDATE: c̃(t) = tanh(Wc · [h(t-1), x(t)] + bc) Creates candidate values to add to state 4. NEW CELL: c(t) = f(t) * c(t-1) + i(t) * c̃(t) Old state × forget + new candidates × input gate 5. OUTPUT GATE: o(t) = σ(Wo · [h(t-1), x(t)] + bo) Decides what to output 6. HIDDEN STATE: h(t) = o(t) * tanh(c(t)) Final output of this cell

Input Features

The LSTM receives a feature vector from each candle containing normalized market data:

LSTM INPUT FEATURES (per candle) ═══════════════════════════════════════════════════════════════════════════════ Feature │ Normalization │ Range ─────────────────────┼──────────────────────────────┼────────────────── Price │ (price - mean) / std │ ~ -3 to +3 VWAP Distance │ (price - vwap) / atr │ ~ -5 to +5 EMA(9) Distance │ (price - ema9) / atr │ ~ -5 to +5 EMA(21) Distance │ (price - ema21) / atr │ ~ -5 to +5 EMA(50) Distance │ (price - ema50) / atr │ ~ -5 to +5 RSI │ (rsi - 50) / 50 │ -1 to +1 MACD Histogram │ histogram / atr │ ~ -3 to +3 ADX │ adx / 100 │ 0 to 1 +DI - -DI │ (plusDI - minusDI) / 100 │ -1 to +1 Volume Ratio │ volume / avgVolume - 1 │ ~ -1 to +5 ─────────────────────┴──────────────────────────────┴────────────────── Total: 10 features per time step × N time steps = Input tensor

LSTM Prediction Output

LSTM PREDICTION STRUCTURE ═══════════════════════════════════════════════════════════════════════════════ LstmPrediction { Direction // -1.0 (bearish) to +1.0 (bullish) Confidence // 0.0 to 1.0 (how sure is the prediction) PredictedChangePercent // Expected price change % PredictedVolatility // Expected volatility ScoreAdjustment // -25 to +25 points to apply to market score IsUsable // True if enough data for reliable prediction } SCORE ADJUSTMENT CALCULATION ═══════════════════════════════════════════════════════════════════════════════ ScoreAdjustment = Direction × Confidence × MaxAdjustment Example: ───────────────────────────────────────────────────────────────────────── Direction = +0.7 (bullish) Confidence = 0.8 (high confidence) MaxAdjustment = 25 ScoreAdjustment = 0.7 × 0.8 × 25 = +14 points If base Market Score was +55: Adjusted Score = 55 + 14 = +69 (closer to entry threshold!)

Effect on TP/SL Sizing

LSTM volatility predictions influence how AdaptiveOrder sizes take profit and stop loss levels.

VOLATILITY-BASED TP/SL ADJUSTMENT ═══════════════════════════════════════════════════════════════════════════════ LSTM Predicts High Volatility + High Confidence: ┌─────────────────────────────────────────────────────────────────────────┐ │ Take Profit: Extended more aggressively (expecting larger moves) │ │ Stop Loss: Widened to avoid noise stops │ └─────────────────────────────────────────────────────────────────────────┘ LSTM Predicts Low Volatility: ┌─────────────────────────────────────────────────────────────────────────┐ │ Take Profit: Kept conservative (smaller expected moves) │ │ Stop Loss: Tightened (less noise expected) │ └─────────────────────────────────────────────────────────────────────────┘ VISUALIZATION ═══════════════════════════════════════════════════════════════════════════════ HIGH VOLATILITY PREDICTION LOW VOLATILITY PREDICTION TP ────────────────── $165 TP ────── $158 Extended (+50%) Normal Entry ═══════════════ $150 Entry ═══ $150 Widened (+30%) Tight SL ────────────────── $143 SL ────── $147
