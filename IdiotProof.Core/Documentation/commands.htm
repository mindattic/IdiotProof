Command Reference

Complete reference for all IdiotScript commands, with signatures and examples.

Configuration Commands

These commands set up the strategy. Order doesn't matter within this group.

Command Description Example
Ticker(symbol) Sets the stock symbol. Required. Ticker("AAPL") or Ticker(AAPL)
Name(name) Friendly name for the strategy. Name("AAPL VWAP Bounce")
Session(session) Trading session: IS.PREMARKET, IS.RTH, IS.AFTERHOURS, IS.EXTENDED Session(IS.RTH)
Quantity(shares) Number of shares per order. Alias: Qty() Quantity(100)

Entry Condition Commands

Conditions that must be met before order execution. Evaluated sequentially.

Price-Based Conditions

Command Description Example
Entry(price) Price must reach this level. Alias: IsPriceAbove() Entry(150.00)
Breakout(price) Price must break above level with momentum. Breakout(155.00)
Pullback(price) Price must pull back to level from above. Pullback(148.50)

VWAP Conditions

Command Description
IsAboveVwap() Current price is above VWAP.
IsBelowVwap() Current price is below VWAP.
IsCloseAboveVwap() Candle closed above VWAP (stronger signal).
IsVwapRejection() Wick above VWAP but close below (bearish rejection).

EMA Conditions

Command Description Example
IsEmaAbove(period) Price is above EMA of specified period. IsEmaAbove(9)
IsEmaBelow(period) Price is below EMA of specified period. IsEmaBelow(21)
IsEmaBetween(p1, p2) Price is between two EMAs (pullback zone). IsEmaBetween(9, 21)
IsEmaTurningUp(period) EMA slope is turning positive. IsEmaTurningUp(9)

Momentum Conditions

Command Description Example
IsMomentumAbove(value) Momentum >= threshold. IsMomentumAbove(0)
IsMomentumBelow(value) Momentum <= threshold. IsMomentumBelow(-5)
IsRocAbove(pct) Rate of Change >= percentage. IsRocAbove(2)
IsRocBelow(pct) Rate of Change <= percentage. IsRocBelow(-2)

Trend/Strength Conditions

Command Description Example
IsAdxAbove(value) ADX >= threshold (trend strength). IsAdxAbove(25)
IsDiPositive() +DI > -DI (bullish directional movement). IsDiPositive()
IsDiNegative() -DI > +DI (bearish directional movement). IsDiNegative()
IsMacdBullish() MACD > Signal line. IsMacdBullish()
IsMacdBearish() MACD < Signal line. IsMacdBearish()

RSI Conditions

Command Description Example
IsRsiOversold(value) RSI <= threshold. IsRsiOversold(30)
IsRsiOverbought(value) RSI >= threshold. IsRsiOverbought(70)

Pattern/Volume Conditions

Command Description Example
IsHigherLows() Higher lows pattern forming. IsHigherLows()
IsVolumeAbove(mult) Volume >= multiplier × average. IsVolumeAbove(1.5)
IsGapUp(pct) Gapped up X% from previous close. IsGapUp(5)
IsGapDown(pct) Gapped down X% from previous close. IsGapDown(3)

Order Commands

Command Description Example
Order() Execute order. Default: LONG. Order(IS.LONG)
Long() Alias for Order(IS.LONG). Long()
Short() Alias for Order(IS.SHORT). Short()
CloseLong() Create SELL order to close long position. CloseLong()
CloseShort() Create BUY order to cover short position. CloseShort()

Order Modifiers

Command Description Example
PriceType(type) Price type for execution: Price.Current, Price.VWAP PriceType(Price.Current)
OrderType(type) Market vs Limit: OrderType.Market, OrderType.Limit OrderType(OrderType.Market)
OutsideRTH() Allow entry order outside RTH. Default: true. OutsideRTH()
TakeProfitOutsideRTH() Allow TP order outside RTH. Default: true. TakeProfitOutsideRTH()

Exit Commands

Command Description Example
TakeProfit(price) Exit at profit target. Alias: TP() TakeProfit(160)
StopLoss(price) Exit at stop level. Alias: SL() StopLoss(145)
TrailingStopLoss(pct) Trailing stop percentage. Alias: TSL() TrailingStopLoss(2)
ExitStrategy(time) Exit at specific time or IS.BELL. ExitStrategy(IS.BELL)
IsProfitable() Chain to only exit if profitable. ExitStrategy(IS.BELL).IsProfitable()

AI & Automation Commands

Command Description Example
AdaptiveOrder() Enable dynamic TP/SL adjustment. AdaptiveOrder(IS.AGGRESSIVE)
AutonomousTrading() Enable fully automated AI trading. AutonomousTrading()
Repeat() Restart strategy after exit. Repeat()

Constants Reference

Session Constants

Constant Description Hours (ET)
IS.PREMARKET Premarket session 4:00 AM - 9:30 AM
IS.RTH Regular Trading Hours 9:30 AM - 4:00 PM
IS.AFTERHOURS After Hours 4:00 PM - 8:00 PM
IS.EXTENDED All hours combined 4:00 AM - 8:00 PM
IS.BELL 1 min before session end Varies by session

Direction Constants

Constant Description
IS.LONG Long position (buy)
IS.SHORT Short position (sell)

Mode Constants

Constant Description
IS.CONSERVATIVE Risk-averse mode
IS.BALANCED Standard mode
IS.AGGRESSIVE Profit-seeking mode
