Submit Ideas

Have an idea for IdiotProof? Add it to this wishlist or discuss it with the development team.

When proposing a feature, consider:

  • What problem does it solve?
  • How would it integrate with existing IdiotScript commands?
  • What are the edge cases to consider?
  • Is there a simpler way to achieve the same goal?
