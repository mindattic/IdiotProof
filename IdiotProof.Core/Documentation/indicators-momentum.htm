Momentum Indicators

Momentum indicators measure the rate of price change, helping identify trend strength, potential reversals, and overbought/oversold conditions.

Momentum Indicators in IdiotProof

IdiotProof uses several momentum indicators to measure the speed and strength of price movements. These indicators are crucial for:

  • Confirming trends - Strong momentum validates trend direction
  • Spotting reversals - Divergences warn of potential turns
  • Timing entries - RSI extremes suggest entry/exit points
  • Filtering trades - Avoid entering against momentum

RSI (Relative Strength Index)

RSI measures the speed and magnitude of recent price changes on a scale of 0 to 100. Traditional levels: below 30 = oversold, above 70 = overbought.

RSI SCALE ═══════════════════════════════════════════════════════════════════════════════ 100 ──────────────────────────────────────────────────────── Top │ 70 ═══════════════════════ OVERBOUGHT ═══════════════════════ │ Zone │ ____ ____ │ / \ / \ 50 ───/──────\──────────────────────/──────\───── Neutral │ \ / \ │ \ / \ 30 ═══════════════════════ OVERSOLD ═════════════════════════ │ Zone 0 ──────────────────────────────────────────────────────── Bottom RSI > 70: Potential pullback or reversal (wait to buy) RSI < 30: Potential bounce or reversal (look for entry) RSI ~50: Neutral momentum

RSI IdiotScript Commands

IsRsiOversold(threshold)

Returns true when RSI is at or below the threshold (default: 30).

Ticker(AAPL)
    .IsRsiOversold(30)   // RSI <= 30 (bouncing from oversold)
    .IsAboveVwap()
    .Entry(150)
    .TakeProfit(158)

IsRsiOverbought(threshold)

Returns true when RSI is at or above the threshold (default: 70).

Ticker(NVDA)
    .IsRsiOverbought(70)  // RSI >= 70 (potential short entry)
    .IsBelowVwap()
    .Order(IS.SHORT)
    .TakeProfit(880)

MACD (Moving Average Convergence Divergence)

MACD shows the relationship between two EMAs. It consists of the MACD line, signal line, and histogram showing the difference between them.

MACD COMPONENTS ═══════════════════════════════════════════════════════════════════════════════ MACD Line = EMA(12) - EMA(26) Signal Line = EMA(9) of MACD Line Histogram = MACD Line - Signal Line BULLISH SIGNAL (MACD > Signal) MACD Line ___/ / \ ___/ Signal Line ____________/ \ ___/ \__/ ↑ Bullish crossover Histogram: ████ ████████ ████████████ ████ ───────────────────────────────── Zero Line ████████ ████████████ BEARISH SIGNAL (MACD < Signal) Signal Line ___________ \ ___ \__/ \ ____ \__/ \___ MACD Line \ ↑ Bearish crossover

MACD IdiotScript Commands

IsMacdBullish()

Returns true when MACD line is above the Signal line.

Ticker(AAPL)
    .IsMacdBullish()     // MACD > Signal line
    .IsEmaAbove(9)
    .Entry(150)
    .TakeProfit(158)

IsMacdBearish()

Returns true when MACD line is below the Signal line.

Ticker(SPY)
    .IsMacdBearish()     // MACD < Signal line
    .IsEmaBelow(21)
    .Order(IS.SHORT)
    .TakeProfit(490)

ADX and Directional Indicators (DI)

ADX measures trend strength (not direction), while +DI and -DI indicate bullish and bearish directional movement.

ADX TREND STRENGTH SCALE ═══════════════════════════════════════════════════════════════════════════════ ADX Value │ Trend Strength │ Trading Implication ──────────────┼──────────────────────────┼───────────────────────────────── 0-20 │ Weak/No Trend │ Range-bound, mean reversion 20-25 │ Trend Developing │ Watch for breakouts 25-50 │ Strong Trend │ Trade with the trend 50-75 │ Very Strong Trend │ Strong continuation expected 75+ │ Extremely Strong │ Rare, usually precedes reversal DIRECTIONAL INDICATORS (+DI / -DI) ═══════════════════════════════════════════════════════════════════════════════ BULLISH: +DI > -DI +DI ────────────────────────/─────────────── / / ────────────/──────── -DI Higher highs dominating = Bullish pressure BEARISH: -DI > +DI ────────────\──────── -DI \ \ +DI ────────────────────────\─────────────── Lower lows dominating = Bearish pressure DI CROSSOVER +DI crosses above -DI ↓ +DI ──────────────────X─────────────────── /│ / │ ← Buy signal (with ADX > 25) -DI ───────────────/──X────────────────── │ -DI crosses below +DI

ADX/DI IdiotScript Commands

IsAdxAbove(threshold)

Returns true when ADX is above the threshold, indicating strong trend.

Ticker(TSLA)
    .IsAdxAbove(25)       // Strong trend exists
    .IsDiPositive()       // Bullish direction
    .Entry(250)
    .TakeProfit(265)

IsDiPositive()

Returns true when +DI is greater than -DI (bullish directional movement).

Ticker(AAPL)
    .IsDiPositive()       // +DI > -DI (bullish)
    .IsAdxAbove(20)
    .IsEmaAbove(9)
    .Entry(150)

IsDiNegative()

Returns true when -DI is greater than +DI (bearish directional movement).

Ticker(SPY)
    .IsDiNegative()       // -DI > +DI (bearish)
    .IsAdxAbove(25)
    .Order(IS.SHORT)
    .TakeProfit(495)

Momentum and Rate of Change (ROC)

These are simpler momentum measures based on price change over a period.

MOMENTUM & ROC CALCULATIONS ═══════════════════════════════════════════════════════════════════════════════ Momentum = Current Price - Price N bars ago ROC = ((Current Price - Price N bars ago) / Price N bars ago) × 100 Example (10-bar period): Current Price: $52.00 Price 10 bars ago: $50.00 Momentum = $52.00 - $50.00 = +$2.00 ROC = (($52 - $50) / $50) × 100 = +4.0% MOMENTUM CHART ═══════════════════════════════════════════════════════════════════════════════ Momentum ↑ │ _____ │ _____/ \____ ← Positive momentum │ / \ ──┼──────────────────────\────────────── Zero line │ \___ │ \___ ← Negative momentum │ └───────────────────────────────────▶ Time Above zero = Bullish momentum Below zero = Bearish momentum Crossing zero = Potential trend change

Momentum/ROC IdiotScript Commands

IsMomentumAbove(threshold)

Returns true when momentum is above the threshold (usually 0).

Ticker(NVDA)
    .IsMomentumAbove(0)   // Positive momentum
    .IsAboveVwap()
    .Entry(900)
    .TakeProfit(920)

IsMomentumBelow(threshold)

Returns true when momentum is below the threshold.

Ticker(SPY)
    .IsMomentumBelow(0)   // Negative momentum
    .Order(IS.SHORT)
    .TakeProfit(495)

IsRocAbove(percent)

Returns true when Rate of Change is above the percentage threshold.

Ticker(TSLA)
    .IsRocAbove(2)       // ROC > 2% (strong upward acceleration)
    .IsEmaAbove(9)
    .Entry(250)
    .TakeProfit(275)

IsRocBelow(percent)

Returns true when Rate of Change is below the percentage threshold.

Ticker(SPY)
    .IsRocBelow(-2)      // ROC < -2% (strong downward acceleration)
    .Order(IS.SHORT)
    .TakeProfit(490)

Momentum in Market Score

Momentum indicators have significant weight in the market score calculation:

Indicator Weight Score Contribution
MACD 18% MACD > Signal: +50 base + histogram strength
ADX/DI 15% ±(ADX value × direction)
RSI 12% Oversold (+100) to Overbought (-100)

Combined, momentum indicators account for 45% of the total market score, making them the most influential category.

Warm-Up Requirements

Indicator Bars Needed Start Early By
RSI(14) 15 bars 20 minutes
MACD(12,26,9) 35 bars 40 minutes
ADX(14) / DI 28 bars 30 minutes
Momentum(10) 11 bars 15 minutes
ROC(10) 11 bars 15 minutes
⚠️ MACD Warm-Up

MACD requires the most warm-up time (35+ bars) because it uses EMA(26) in its calculation. Start the backend 40+ minutes before trading if using MACD signals.

Momentum Trading Strategies

1. RSI Bounce Strategy

Enter when RSI bounces from oversold with trend confirmation.

Ticker(AAPL)
    .Name("RSI Bounce")
    .Session(IS.RTH)
    .Quantity(50)
    .IsRsiOversold(30)     // RSI bouncing from 30
    .IsAboveVwap()          // Price reclaimed VWAP
    .IsMomentumAbove(0)     // Momentum turning positive
    .Entry(150)
    .TakeProfit(158)
    .StopLoss(146)

2. MACD Momentum Breakout

Trade breakouts with MACD confirmation.

Ticker(NVDA)
    .Name("MACD Momentum")
    .Session(IS.RTH)
    .Quantity(20)
    .IsMacdBullish()        // MACD > Signal
    .IsAdxAbove(25)         // Strong trend
    .IsDiPositive()         // Bullish direction
    .IsEmaAbove(9)          // Above short-term EMA
    .Entry(900)
    .TakeProfit(930)
    .StopLoss(880)
    .AdaptiveOrder(IS.AGGRESSIVE)

3. Trend Strength Filter

Only trade when ADX confirms a strong trend.

Ticker(TSLA)
    .Name("ADX Trend Filter")
    .Session(IS.RTH)
    .Quantity(15)
    .IsAdxAbove(30)         // Very strong trend
    .IsDiPositive()         // Bullish
    .IsMacdBullish()        // MACD confirms
    .IsEmaAbove(21)         // Above trend EMA
    .Entry(250)
    .TakeProfit(275)
    .StopLoss(240)

4. Momentum Short

Short when all momentum indicators are bearish.

Ticker(SPY)
    .Name("Momentum Short")
    .Session(IS.RTH)
    .Quantity(100)
    .IsMacdBearish()        // MACD < Signal
    .IsDiNegative()         // -DI > +DI
    .IsAdxAbove(25)         // Strong trend
    .IsMomentumBelow(0)     // Negative momentum
    .IsBelowVwap()          // Below fair value
    .Order(IS.SHORT)
    .TakeProfit(490)
    .StopLoss(508)

Momentum Trading Tips

💡 Pro Tips
  • Combine ADX with DI - ADX alone doesn't show direction
  • RSI extremes work better in ranges - In strong trends, RSI can stay overbought
  • Watch for MACD histogram - Shrinking histogram warns of momentum loss
  • ADX > 25 filters out choppy trades - Only trade with confirmed trends
  • Look for divergences - Price making new highs while RSI doesn't = warning
⚠️ Common Mistakes
  • Shorting just because RSI is overbought - strong stocks stay overbought
  • Ignoring ADX strength when trading MACD crossovers
  • Using momentum indicators in choppy/ranging markets
  • Not waiting for warm-up period (especially MACD)
