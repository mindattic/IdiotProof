Technical Indicators

A comprehensive guide to all 11 technical indicators used by IdiotProof's Market Score system, with ASCII visualizations and practical usage.

Indicator Warm-Up Requirements

Technical indicators require historical price bars to calculate properly. The backend uses 1-minute OHLC bars. You must start the backend early enough to collect sufficient bars before trading.

🚨 Start Early!

During warm-up, indicator conditions will NOT trigger. This prevents false entries based on incomplete data.

VWAP (Volume-Weighted Average Price)

VWAP represents the average price weighted by volume throughout the day. It's a key institutional benchmark - price above VWAP suggests buying pressure, below suggests selling pressure.

VWAP VISUALIZATION ═══════════════════════════════════════════════════════════════════════════════ Price ↑ │ ___/\ │ ___/\ ___--- \___ │ ___-/ \___ ___-/ \ │ ___-/ \/ \ ══╪════════════════════════════════════════════════════ VWAP │ _/ \_____ │/ │ └────────────────────────────────────────────────────→ Time 9:30 11:00 13:00 15:00 ▲ ABOVE VWAP ▼ BELOW VWAP Bullish Bearish VWAP REJECTION PATTERN ═══════════════════════════════════════════════════════════════════════════════ │ ← Wick tests above VWAP VWAP ═══╪════════════════════════════════════════════════════ ┌─┴─┐ │ │ ← Close BELOW VWAP = REJECTED │ │ Bearish signal └───┘ IsVwapRejection() detects this pattern

IdiotScript Commands

Command Description
IsAboveVwap() Current price is above VWAP
IsBelowVwap() Current price is below VWAP
IsCloseAboveVwap() Candle CLOSED above VWAP (stronger)
IsVwapRejection() Wick above VWAP, close below (bearish)

EMA (Exponential Moving Average)

EMAs smooth price action and identify trends. IdiotProof uses a stack of EMAs (9, 21, 50) to determine trend strength and alignment.

EMA STACK VISUALIZATION ═══════════════════════════════════════════════════════════════════════════════ BULLISH ALIGNMENT (Price > EMA9 > EMA21 > EMA50) Price ─────────────────────────────/ __/ EMA(9) ──────────────────────__/ __/ EMA(21) ───────────────__/ __/ EMA(50) ────────__/ BEARISH ALIGNMENT (Price < EMA9 < EMA21 < EMA50) EMA(50) ────────\__ \__ EMA(21) ───────────────\__ \__ EMA(9) ──────────────────────\__ \ Price ─────────────────────────────\ EMA PULLBACK ZONE ═══════════════════════════════════════════════════════════════════════════════ Price pulls back ↓ /\ /\ / \ / \____ Price / \ / EMA(9) ──────────────────────────── \ / \______/ ← Pullback zone EMA(21) ─────────────────────────── IsEmaBetween(9, 21) - Price in pullback zone EMA TURNING UP PATTERN ═══════════════════════════════════════════════════════════════════════════════ ___/ ___/ ___/ ← EMA slope turning positive \_____ __/ \______________/ ↑ Flattening before turn IsEmaTurningUp(9) - EMA9 slope positive or flat

IdiotScript Commands

Command Description
IsEmaAbove(9) Price is above EMA(9)
IsEmaBelow(21) Price is below EMA(21)
IsEmaBetween(9, 21) Price between EMA(9) and EMA(21)
IsEmaTurningUp(9) EMA(9) slope is turning positive

RSI (Relative Strength Index)

RSI measures the speed and magnitude of price movements on a 0-100 scale. It identifies overbought (>70) and oversold (<30) conditions.

RSI SCALE AND ZONES ═══════════════════════════════════════════════════════════════════════════════ 0 30 50 70 100 │ │ │ │ │ ├──────────┼──────────┼──────────┼──────────┤ │▓▓▓▓▓▓▓▓▓▓│ │ │▓▓▓▓▓▓▓▓▓▓│ │ OVERSOLD │ │ │OVERBOUGHT│ └──────────┴──────────┴──────────┴──────────┘ ↑ ↑ BUY SIGNAL SELL SIGNAL (Mean reversion) (Mean reversion) RSI DIVERGENCE ═══════════════════════════════════════════════════════════════════════════════ BULLISH DIVERGENCE (Price falls, RSI rises) Price: ╲ ╲___ ╲___ ← Price makes lower lows RSI: ___/ ___/ ___/ ← RSI makes higher lows → Momentum building, potential reversal UP BEARISH DIVERGENCE (Price rises, RSI falls) Price: ___/ ___/ ___/ ← Price makes higher highs RSI: ╲ ╲___ ╲ ← RSI makes lower highs → Momentum fading, potential reversal DOWN

IdiotScript Commands

Command Description
IsRsiOversold(30) RSI ≤ 30 (oversold, potential bounce)
IsRsiOverbought(70) RSI ≥ 70 (overbought, potential pullback)

MACD (Moving Average Convergence Divergence)

MACD shows the relationship between two EMAs. It consists of the MACD line (EMA12-EMA26), Signal line (EMA9 of MACD), and Histogram (MACD - Signal).

IdiotScript Commands

Command Description
IsMacdBullish() MACD > Signal line
IsMacdBearish() MACD < Signal line

ADX (Average Directional Index)

ADX measures trend strength (not direction) on a 0-100 scale. +DI and -DI indicate directional movement.

ADX TREND STRENGTH ═══════════════════════════════════════════════════════════════════════════════ ADX Value │ Trend Strength │ Trading Implication ──────────┼─────────────────┼──────────────────────────────────── 0-20 │ Weak/No Trend │ Range-bound, avoid trend strategies 20-25 │ Developing │ Trend may be starting 25-50 │ Strong │ Good for trend-following 50-75 │ Very Strong │ Excellent trend 75-100 │ Extremely Strong│ Rare, potential exhaustion ADX WITH DIRECTIONAL INDICATORS ═══════════════════════════════════════════════════════════════════════════════ +DI (bullish) / \ ___/ / \ ___/ \ / \__/ \ / ___ \____ ___/__/ \_____ \ -DI (bearish) ↑ +DI > -DI = Bullish (IsDiPositive) When ADX rising + DI positive crossover = Strong bullish trend confirmed When ADX rising + DI negative crossover = Strong bearish trend confirmed When ADX falling = Trend weakening, prepare for range/reversal

IdiotScript Commands

Command Description
IsAdxAbove(25) ADX ≥ 25 (strong trend exists)
IsDiPositive() +DI > -DI (bullish movement)
IsDiNegative() -DI > +DI (bearish movement)

Volume Analysis

Volume confirms price moves. High volume on up-moves is bullish; high volume on down-moves is bearish. The system compares current volume to a 20-period average.

VOLUME CONFIRMATION ═══════════════════════════════════════════════════════════════════════════════ ████ ← Volume spike (1.5x+ average) ████ Confirms the move! Average ─────────────────────────── ████ ████████ ████ ████ ██ ████████ ██ ████ ──── ── ──────── ── ──── Low vol = Weak move VOLUME PATTERNS ═══════════════════════════════════════════════════════════════════════════════ HEALTHY UPTREND: • Price rising + Volume rising = Strong • Price rising + Volume falling = Weakening HEALTHY DOWNTREND: • Price falling + Volume rising = Strong selling • Price falling + Volume falling = Selling exhaustion REVERSAL SIGNALS: • Climax volume at top/bottom = Potential reversal

IdiotScript Commands

Command Description
IsVolumeAbove(1.5) Current volume ≥ 1.5× average
