What is IdiotProof?

IdiotProof is a sophisticated algorithmic trading system that combines multiple technical indicators, machine learning models, and AI-powered analysis to make autonomous trading decisions. Built on top of the Interactive Brokers TWS API, it provides:

📜
IdiotScript DSL
A domain-specific language for defining trading strategies in plain, readable syntax. Write Ticker(AAPL).Entry(150).TakeProfit(160) and let the system handle the rest.
🎯
Market Score System
Combines 11 technical indicators into a single score from -100 (bearish) to +100 (bullish). Uses learned weights optimized per ticker.
🤖
Autonomous Trading
Self-adjusting entry/exit thresholds based on real-time market conditions. The system adapts to trending vs ranging markets automatically.
🧠
AI Integration
LSTM neural networks for price prediction, LSH pattern matching for historical analogs, and ChatGPT-powered decision support.
🔄
Adaptive Orders
Dynamic take profit and stop loss adjustments based on momentum. Extends targets in strong trends, contracts in weak conditions.
📊
Comprehensive Analysis
Backtesting, Monte Carlo simulation, walk-forward analysis, and real-time performance tracking built in.

Quick Start

1. Define a Strategy

Create an IdiotScript strategy using the fluent API:

Ticker(NVDA)
    .Name("NVDA Breakout")
    .Session(IS.RTH)
    .Quantity(10)
    .Entry(850)
    .IsAboveVwap()
    .IsEmaAbove(9)
    .TakeProfit(880)
    .StopLoss(835)
    .AdaptiveOrder(IS.BALANCED)

2. Or Use Autonomous Trading

Let the AI handle everything:

Ticker(AAPL)
    .Name("AAPL Auto")
    .Session(IS.RTH)
    .Quantity(25)
    .AutonomousTrading()

3. Start the Backend

⚠️ Indicator Warm-Up Required

Technical indicators require historical price bars to calculate properly. Start the backend 30+ minutes before trading to allow all indicators to warm up. See the Indicator Warm-Up Guide.

For complete setup instructions, see the Quick Start Guide.
