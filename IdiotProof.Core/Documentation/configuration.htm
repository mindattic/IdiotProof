Configuration Settings

Configure IdiotProof connection, trading, and AI settings.

Configuration Files

CONFIGURATION FILES ═══════════════════════════════════════════════════════════════════════════════ IdiotProof.Core/ │ ├── Settings/ │ └── settings.json ← Main application settings │ ├── Scripts/ │ (watchlist.json moved to Data/ folder) │ └── *.idiot ← Individual strategy files │ └── Data/ └── strategy-rules.json ← Custom AI evaluation rules

settings.json

Main configuration file for the application.

Complete Example

{ "ibkr": { "host": "127.0.0.1", "port": 7497, "clientId": 1, "accountId": "" }, "trading": { "defaultQuantity": 100, "defaultSession": "RTH", "allowOutsideRth": true, "paperTrading": true }, "ai": { "openaiApiKey": "sk-...", "model": "gpt-4", "enableAdvisor": true, "enableLstm": true, "enablePatternMatcher": true }, "logging": { "level": "Info", "writeToFile": true, "writeToConsole": true }, "indicators": { "emaPeriods": [9, 21, 50, 200], "rsiPeriod": 14, "adxPeriod": 14, "macdFast": 12, "macdSlow": 26, "macdSignal": 9 } }

IBKR Connection Settings

Field Type Default Description
host string "127.0.0.1" TWS host address (usually localhost)
port integer 7497 TWS port: 7497 (paper) or 7496 (live)
clientId integer 1 Unique client ID (change if multiple connections)
accountId string "" Specific account ID (blank for default)
PORT CONFIGURATION ═══════════════════════════════════════════════════════════════════════════════ TWS Connection Ports: ┌─────────────────────────────────────────────────────────────────────────┐ │ Port 7496 │ LIVE Trading │ Real money - use with caution! │ ├─────────────────────────────────────────────────────────────────────────┤ │ Port 7497 │ PAPER Trading │ Simulated - safe for testing │ └─────────────────────────────────────────────────────────────────────────┘ Recommended: Start with port 7497 (paper) until comfortable.

Trading Settings

Field Type Default Description
defaultQuantity integer 100 Default shares per order if not specified in strategy
defaultSession string "RTH" Default session: RTH, Premarket, AfterHours, Extended
allowOutsideRth boolean true Allow orders outside regular trading hours
paperTrading boolean true Safety flag - must be false for live trading

AI Settings

Field Type Default Description
openaiApiKey string "" OpenAI API key for AI Advisor (sk-...)
model string "gpt-4" OpenAI model to use (gpt-4, gpt-3.5-turbo)
enableAdvisor boolean true Enable ChatGPT-based AI Advisor
enableLstm boolean true Enable LSTM neural network predictions
enablePatternMatcher boolean true Enable LSH pattern matching

Note: You can also set the API key via environment variable:

$env:OPENAI_API_KEY = "sk-..."

Logging Settings

Field Type Default Description
level string "Info" Log level: Debug, Info, Warn, Error
writeToFile boolean true Write logs to Logs/session_*.txt
writeToConsole boolean true Write logs to console output

Indicator Settings

Field Type Default Description
emaPeriods array [9, 21, 50, 200] EMA periods to calculate
rsiPeriod integer 14 RSI calculation period
adxPeriod integer 14 ADX/DI calculation period
macdFast integer 12 MACD fast EMA period
macdSlow integer 26 MACD slow EMA period
macdSignal integer 9 MACD signal line period

Environment Variables

These environment variables can override settings.json:

Variable Description
OPENAI_API_KEY OpenAI API key for AI Advisor
IDIOTPROOF_IBKR_PORT Override IBKR connection port
IDIOTPROOF_LOG_LEVEL Override log level
