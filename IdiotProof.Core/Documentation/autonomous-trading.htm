AutonomousTrading

Fully automated AI-driven trading where the system independently decides when to buy, sell, short, or close positions based on real-time market analysis.

Quick Start

// Minimal autonomous trading setup Ticker(AAPL) .AutonomousTrading() // That's it! The system will: // 1. Monitor VWAP, EMA, RSI, MACD, ADX, and Volume continuously // 2. Calculate a market score (-100 to +100) // 3. Self-adjust thresholds based on market conditions // 4. Enter LONG when score >= dynamic threshold // 5. Enter SHORT when score <= negative threshold // 6. Exit when momentum reverses // 7. Flip direction on strong reversals
💡 Recommended Setup
Ticker(NVDA) .Name("NVDA Autonomous") .Session(IS.RTH) .Quantity(5) .AutonomousTrading()

Self-Adjusting Thresholds

The system automatically adjusts entry/exit thresholds based on real-time market conditions. There are no modes to configure - it's always optimal.

Automatic Take Profit & Stop Loss

AutonomousTrading automatically calculates TP and SL based on ATR (Average True Range), with multipliers that adapt to market conditions.

ATR-BASED TP/SL CALCULATION ═══════════════════════════════════════════════════════════════════════════════ STRONG TREND (ADX >= 40): ├── TP multiplier: 2.5× ATR (let winners run) ├── SL multiplier: 1.2× ATR (protect capital) └── Example: ATR = $2.00 → TP = Entry + $5.00 → SL = Entry - $2.40 MODERATE TREND (ADX 25-40): ├── TP multiplier: 2.2× ATR ├── SL multiplier: 1.4× ATR └── Example: ATR = $2.00 → TP = Entry + $4.40 → SL = Entry - $2.80 RANGING MARKET (ADX < 20): ├── TP multiplier: 1.5× ATR (take quick profits) ├── SL multiplier: 2.0× ATR (allow for chop) └── Example: ATR = $2.00 → TP = Entry + $3.00 → SL = Entry - $4.00

Ticker Learning System

AutonomousTrading builds a TickerProfile for each symbol over time, learning what works specifically for that stock.

TICKER LEARNING SYSTEM ═══════════════════════════════════════════════════════════════════════════════ WHAT IT LEARNS: ┌───────────────────────────────────────────────────────────────────────┐ │ Optimal entry thresholds - Score level with best win rate │ │ Optimal exit thresholds - When to exit for max profit │ │ Time-of-day patterns - Best hours/minutes to trade │ │ Indicator correlations - Which signals work best │ │ Win rate at different levels - Bucketed by entry score │ │ Streak awareness - More conservative after losses │ └───────────────────────────────────────────────────────────────────────┘ HOW IT ADAPTS: ┌───────────────────────────────────────────────────────────────────────┐ │ 1. AutonomousTrading vs AdaptiveOrder

Feature AutonomousTrading AdaptiveOrder
Entry Decision AUTOMATIC (AI decides) MANUAL (you set Entry)
Exit Decision AUTOMATIC (AI decides) SEMI-AUTO (adjusts TP/SL)
TP/SL Calculation ATR-based automatic Modifies your TP/SL
Direction Can flip long↔short Single direction only
Conditions Required NONE (just Ticker) You define conditions
Best For Fully hands-off Enhance manual strategy
When to Use Which?
  • AutonomousTrading: You want the AI to handle everything
  • AdaptiveOrder: You have specific entry conditions but want smart exits
