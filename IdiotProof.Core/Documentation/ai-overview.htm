AI & Machine Learning

IdiotProof integrates multiple AI technologies to enhance trading decisions, from LSTM neural networks to pattern matching and ChatGPT advisory.

The Three AI Components

🔮

LSTM Predictor

Long Short-Term Memory neural network that predicts price direction. Captures temporal patterns in sequential data.

  • Adjusts market score by ±25 points
  • Predicts volatility for TP/SL sizing
  • Trained on ticker-specific data
Learn More →
🔍

Pattern Matcher

LSH (Locality-Sensitive Hashing) system that finds historical price patterns similar to current conditions.

  • Sub-millisecond pattern lookup
  • 180-minute rolling window
  • Learns from historical outcomes
Learn More →
💬

AI Advisor (ChatGPT)

ChatGPT integration providing natural language analysis of market conditions and custom strategy rules.

  • Interprets natural language stock tips
  • Returns LONG/SHORT/WAIT decisions
  • Provides reasoning and confidence
Learn More →
📚

Ticker Learning

Per-ticker profile system that learns what works for each stock. Tracks every trade and builds specialized knowledge.

  • Optimal entry/exit thresholds
  • Best time-of-day windows
  • Indicator correlation analysis
Learn More →

AI Data Persistence

All AI learning is persisted to disk and accumulates across trading sessions.

File Contents Location
{TICKER}.weights.json Learned indicator weights Data/
{TICKER}.patterns.json Historical pattern database Data/
{TICKER}.history.json Trade history for learning Data/
{TICKER}.profile.json Ticker-specific preferences Data/
lsh_hyperplanes.json LSH hash functions Data/
