Strategy Examples

Real-world IdiotScript strategies from simple setups to advanced multi-condition entries.

1. VWAP Bounce Strategy

A classic mean-reversion strategy that buys when price reclaims VWAP with confirmation from EMA and volume.

// VWAP Bounce Strategy // Enters when price bounces off VWAP with confirmation Ticker(AAPL) .Name("AAPL VWAP Bounce") .Session(IS.RTH) .Quantity(100) // Entry conditions (sequential) .IsCloseAboveVwap() // Candle must CLOSE above VWAP .IsEmaAbove(9) // Price above short-term EMA .IsVolumeAbove(1.5) // Volume 1.5x average confirms // Order .Order(IS.LONG) // Exits .TakeProfit(175) .StopLoss(165) .TrailingStopLoss(2) // $2 trailing stop

2. Gap and Go Strategy

Catches momentum on stocks that gap up significantly at open.

// Gap and Go - Momentum Strategy // Catches strong gap-up momentum plays Ticker(TSLA) .Name("TSLA Gap and Go") .Session(IS.PREMARKET) // Start in premarket .Quantity(50) // Gap detection .IsGapUp(5) // 5%+ gap from previous close // Confirmation .IsAboveVwap() // Holding above VWAP .IsDiPositive() // Bullish directional movement .IsMomentumAbove(0) // Positive momentum // Aggressive entry for momentum .Order(IS.LONG) // Tight risk management .TakeProfit(280) .StopLoss(255) .AdaptiveOrder(IS.AGGRESSIVE) // Let winners run

3. Trend Following Strategy

A multi-indicator trend confirmation strategy using ADX for trend strength.

// Trend Following Strategy // Multiple indicators confirm strong trend Ticker(NVDA) .Name("NVDA Trend Follow") .Session(IS.RTH) .Quantity(25) // Trend strength check .IsAdxAbove(25) // Strong trend exists .IsDiPositive() // Bullish direction // EMA stack confirmation .IsEmaAbove(9) // Above short-term .IsEmaAbove(21) // Above medium-term // Momentum confirmation .IsMacdBullish() // MACD > Signal .IsMomentumAbove(0) // Positive momentum // Entry and exits .Order(IS.LONG) .TakeProfit(550) .StopLoss(480) .AdaptiveOrder(IS.BALANCED)

4. RSI Mean Reversion Strategy

Buys oversold conditions with pullback confirmation.

// RSI Mean Reversion Strategy // Buy oversold bounces with EMA support Ticker(SPY) .Name("SPY Mean Reversion") .Session(IS.RTH) .Quantity(200) // Oversold detection .IsRsiOversold(30) // RSI <= 30 // Pullback zone .IsEmaBetween(9, 21) // In support zone // Bounce confirmation .IsVolumeAbove(1.5) // Volume on bounce .IsHigherLows() // Building support // Conservative entry .Order(IS.LONG) .TakeProfit(475) .StopLoss(455) .AdaptiveOrder(IS.CONSERVATIVE) // Protect gains

5. Breakout-Pullback Strategy

Waits for breakout above resistance, then enters on pullback.

// Breakout-Pullback Strategy // Breakout above resistance, wait for pullback entry Ticker(AMZN) .Name("AMZN Breakout") .Session(IS.RTH) .Quantity(30) // Breakout detection .Breakout(185) // Price breaks above $185 // Pullback entry .Pullback(182) // Wait for pullback to $182 // Confirmation pullback is holding .IsAboveVwap() // Still above VWAP .IsEmaAbove(9) // EMA support held // Entry .Order(IS.LONG) .TakeProfit(195) .StopLoss(178)

6. Timed Exit Strategy

Uses ExitStrategy to exit at specific time, only if profitable.

// Timed Exit Strategy // Exit at bell if profitable, otherwise hold overnight Ticker(META) .Name("META Timed Exit") .Session(IS.RTH) .Quantity(40) // Entry conditions .IsAboveVwap() .IsEmaAbove(9) .IsMacdBullish() // Entry .Order(IS.LONG) // Standard exits .TakeProfit(620) .StopLoss(580) // Timed exit at bell, only if profitable .ExitStrategy(IS.BELL).IsProfitable()
IS.BELL Values
  • Premarket session: 9:29 AM
  • RTH session: 3:59 PM
  • After Hours session: 7:59 PM

7. Fully Autonomous Trading

Let the AI handle everything - entries, exits, direction.

// Fully Autonomous Trading // AI handles all decisions Ticker(NVDA) .Name("NVDA Autonomous") .Session(IS.RTH) .Quantity(10) .AutonomousTrading() // That's it! The system will: // - Monitor all 11 indicators // - Calculate market score // - Self-adjust entry thresholds // - Enter LONG or SHORT based on score // - Exit when momentum reverses // - Learn and improve over time

8. Repeating Strategy

Strategy that restarts after exit, catching multiple trades per day.

// Repeating Scalp Strategy // Catches multiple VWAP bounces throughout the day Ticker(QQQ) .Name("QQQ Scalper") .Session(IS.RTH) .Quantity(500) // Quick scalp setup .IsCloseAboveVwap() .IsEmaAbove(9) .Order(IS.LONG) // Tight TP/SL for scalping .TakeProfit(455) .StopLoss(450) // Repeat after exit! .Repeat()
